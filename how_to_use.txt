Ознакомимся парой ключевых слов.
```
int, float, string, bolean - тип данных.
~ - однострочный комментарий.
out/in - вывод/ввод.
if/else - условные операторы.
for/while - циклы.
```

1. Вывод
Структура: out выражение;

out "Hello, World!"; ~выведет Hello, World!

2. Переменные.
Структура: 
тип название = значение;
тип название;
тип название1, название2 ... названиеN;

int a = 1; ~создание целочисленной переменной со значением 1

float b = 2.0; ~создание десятичной переменной со значением 2.0

string s = "hi"; ~создание строковой переменной со значением hi.
